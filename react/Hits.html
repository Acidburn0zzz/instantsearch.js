Hits

Displays the list of hits for the current search parameters.

Props

hitsPerPagenumber#

How many hits should be displayed for every page. Ignored when a HitsPerPage component is also present.

Theme

root

Implementing your own Hits

See Making your own widgets for more information on how to use the Hits.connect HOC.

import {Hits} from 'react-instantsearch';

function MyHits(props) {
  return (
    <div>
      {props.hits.map(hit =>
        <HitComponent key={hit.objectID} hit={hit} />
      )}
    </div>
  );
}

// `Hits.connect` accepts the same `hitsPerPage` prop as `Hits`.
export default Hits.connect(MyHits);
