Toggle

The Toggle widget is where users type their search queries.

Props

attributeNamestring#

Name of the attribute on which to apply the value refinement. Required when value is present.

filterfunc#

Custom filter. Takes in a SearchParameters and returns a new SearchParameters with the filter applied.

idstring#

URL state serialization key. The state of this widget takes the form of a string that can be either 'on' or 'off'. Required when attributeName isn't present.

labelstring#

Label for this toggle.

valueany#

Value of the refinement to apply on attributeName. Required when attributeName is present.

Theme

root, checkbox, label

Implementing your own Toggle

See Making your own widgets for more information on how to use the Toggle.connect HOC.

import {Toggle} from 'react-instantsearch';

function MyToggle(props) {
  return (
    <input
      type="checkbox"
      checked={props.checked}
      onChange={e => props.refine(!e.target.value)}
    />
  );
}

// `Toggle.connect` accepts the same `id`, `label`, `filter`, `attributeName`,
// `value` and `defaultChecked` props as `Toggle`.
export default Toggle.connect(MyToggle);
