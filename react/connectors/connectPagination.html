                                    Description

                                    connectPagination connector provides the logic to build a widget that will let the user displays hits corresponding to a certain page.

                                    Exposed props

                                    Props exposed on the connected component:

                                    showFirst
                                    		type: booleandefault: true

                                    Display the first page link.

                                    showLast
                                    		type: booleandefault: false

                                    Display the last page link.

                                    showPrevious
                                    		type: booleandefault: true

                                    Display the previous page link.

                                    showNext
                                    		type: booleandefault: true

                                    Display the next page link.

                                    pagesPadding
                                    		type: numberdefault: 3

                                    How many page links to display around the current page.

                                    maxPages
                                    		type: numberdefault: Infinity

                                    Maximum number of pages to display.

                                    Provided props

                                    These are the props that the wrapped Component will receive.

                                    refine
                                    		type: function 

                                    a function to remove a single filter

                                    createURL
                                    		type: function 

                                    a function to generate a URL for the corresponding search state

                                    nbPages
                                    		type: number 

                                    the total of existing pages

                                    currentRefinement
                                    		type: number 

                                    the page refinement currently applied
